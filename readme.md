# Lets talk about - GraphQL by Tamino

[View Slides](https://graphql.lets-talk-about.tamino.dev)

An introduction to GraphQL — the query language for APIs originally developed by Facebook in 2012 and open-sourced in 2015. The slides cover why GraphQL improves on REST and SQL for API use cases, walk through core concepts like types, scalars, enums, interfaces, unions, schemas, queries, and mutations, and conclude with an overview of Apollo (server, engine, client) including features like subscriptions, caching, and monitoring.
